Tipo de interfaz: interfaz SPI

1.GND = Tierra de energía
2.VCC = La fuente de alimentación es 3.3V
3.SCL = línea de reloj SPI
4.SDA = línea de datos SPI
5.RES = Restablecer pin
6.DC = Pin de control de datos / comando
7.CS = SPI Chip Select
8.BLK = Interruptor de control de retroiluminación de la pantalla
La luz de fondo predeterminada está encendida, la luz baja está apagada

Interruptor de control de retroiluminación; BLK está activado de forma predeterminada;
Cuando BLK está bajo, la luz de fondo está apagada, y también puede pasar
Pwm controla el tiempo de conmutación de BLK para controlar el brillo de la pantalla
